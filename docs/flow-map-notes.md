# Flow Map Notes

Atlas tries to trace where capital starts, how it spreads, and how long it takes to show up elsewhere. The map is more useful when the lag assumptions stay explicit.

## Useful observations

- Origin wallets matter more when downstream activity fans out quickly.
- Propagation lag should be tracked in ranges, not false precision.
- Sector rotation is stronger when related destinations light up in sequence instead of all at once.

## Avoid overstating

- A single bridge deposit is not a trend.
- Shared counterparties can make unrelated flows look connected.
- Late followers should not be mistaken for the origin source.

## Output bias

Prefer explanations that highlight direction and timing over exact attribution claims that the data cannot support.
